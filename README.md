# Spring Security OAuth2 Api 


# Sobre o projeto

Esta API tem como objetivo demonstrar o fluxo de autenticação via OAuth 2.0 com Spring.

# Tecnologias utilizadas
## Back end
- Java 17
- Spring Boot
- JPA / Hibernate
- Spring Security
- JWT
- Mysql
- Maven
